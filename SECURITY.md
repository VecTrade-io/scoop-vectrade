# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest  | :white_check_mark: |
| < latest | :x:               |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

This repository contains a Scoop manifest that installs the VecTrade CLI. If you find a security issue:

- **VecTrade CLI vulnerabilities:** Report to security@vectrade.io
- **Manifest-specific issues:** Report to security@vectrade.io

Include:
- Description of the vulnerability
- Steps to reproduce
- Impact assessment

We will acknowledge receipt within 48 hours.
